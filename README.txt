Generate perfect images for posts, pages or WooCommerce products. Provide a max width and height and this plugin will generate aspect ratio perfect (padded with whitespace) images and thumbnails.

== Description ==

This plugin helps you to generate perfect images for your Posts, Pages and (WooCommerce) Products. 

Generally, you loose image content while uploading due to various sizes and cropping. This plugin, however, while keeping the aspect ratio intact ALSO keeps the image content intact. It does it by adding whitespace padding where ever required. 

You can use this plugin to generate a perfect looking shop or blog with even images sizes.


== Installation ==

1. Upload `wp-perfect-image-cropper-resizer.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
